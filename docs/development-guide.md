# Development Guide

## Prerequisites
- Node.js 20+
- Bun 1.1+ (preferred for scripts)

## Setup
```bash
bun install
```

## Running Locally
```bash
bun dev
```

## Linting & Formatting
```bash
bun run lint
bun run format
```

## Building
```bash
bun run build
bun start
```

## Notes
- Tailwind CSS is configured via `postcss.config.mjs`.
- React Compiler is enabled in `next.config.ts`.
